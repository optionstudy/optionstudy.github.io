## 准备环境
### 0. 电脑最低配置必须满足下列条件，否则软件启动会报错退出
#### a. 内存不小于8GB
#### b. CPU不少于4核
    
#### 查看方法:
#### c. 在windows任务栏上单击右键，选择任务管理器
![任务](img/check.png)
#### d. 在“性能”界面上，CPU的逻辑处理器数量不能小于4，物理内存不能小于8GB
